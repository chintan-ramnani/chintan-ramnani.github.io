# Portfolio Website (GitHub Pages)

This is a lightweight, professional portfolio site (HTML/CSS/JS) made for **Ranjay**.

## 1) Customize your content
Open `profile.js` and edit:
- `tagline`, `about`, `highlights`
- `skills`, `featuredProjects`
- `experience`, `education`
- `contact.email` + `contact.linkedin`
- (Optional) add `resumeUrl` (e.g., `assets/resume.pdf`)

## 2) Deploy on GitHub Pages (no coding tools required)

### A) Create a Pages repo
1. Go to GitHub and **sign in**.
2. Create a new repository named: **ranjay-kum-shan.github.io**
3. Upload these files (keep the folder structure).

### B) Enable Pages
1. Repo → **Settings** → **Pages**
2. Source: **Deploy from a branch**
3. Branch: **main** / root
4. Save

Your site will be available at:
https://ranjay-kum-shan.github.io

## 3) Update later
Just edit `profile.js`, commit, and push.

## Notes
- The Projects section has an **auto GitHub feed** using GitHub’s public API.
- If you have no public repos yet, the feed will be empty — it will automatically populate when you publish repos.
